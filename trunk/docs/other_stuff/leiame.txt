
                      GL Export v1.0
                     ================

1. TERMOS DE USO

Voce pode usar esse script a vontade, contanto que não redistribua como se fosse seu.
Caso voce queira redistribuir, por favor mantenha esse documento junto com a copia.


2. COMO USAR

  Para usar esse script, voce tem que criar um objeto (apenas um) e em seguida abra
a tela de scripts (SHIFT+F11). Aperte o botão "Brose Datablock" (aquele botão com um
traço). E clique em "OPEN NEW", procure o script clique nele para seleciona-lo, após
isso, tecle ENTER.

  O script vai aparecer na tela, e embaixo voce vai poder ver seu nome (logo após TX:)
Quando isso acontecer, execute o script com ALT+P

  Vai aparecer uma tela para voce escolher o nome do arquivo a ser gravado. A primeira
caixa de texto mostra o diretório onde voce está. O segundo é para voce colocar o nome
do arquivo que voce vai exportar. Escolha um e aperte ENTER. Voce não precisa colocar
sua extensão, por que o próprio script faz isso para voce.

  Pronto! Seu arquivo ja foi exportado! Abra o arquivo que voce salvou e veja o codigo.
